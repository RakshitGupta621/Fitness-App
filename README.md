# Fitness-App 
### Running the App

**Note**: Make sure you can [Expo](https://docs.expo.io/ "Expo") installed on your machine.

```
 $ git clone https://github.com/RakshitGupta621/Fitness-App
 $ cd Fitness-App
 $ npm install
 $ expo start
 Press $ i
```
